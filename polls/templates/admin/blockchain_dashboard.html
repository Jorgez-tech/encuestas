üîß Troubleshooting

Starting the Blockchain Network:

  1. Open a terminal and navigate to the blockchain directory
  2. Run npx hardhat node to start the local network
  3. In another terminal, run npx hardhat run scripts/deploy.js --network localhost
  4. Update the contract address in Django settings if needed
  5. Refresh this page

Common Issues:

  • Contract not found: Make sure the contract is deployed and the address is correct
  • Connection failed: Check if Hardhat network is running on the correct port
  • Transaction failed: Ensure the deployer account has sufficient ETH
