🗳️ Sistema de Encuestas Django

📋 Sobre este proyecto

Este es un sistema de encuestas desarrollado con Django que permite crear preguntas, votar y ver resultados en tiempo real. Incluye un panel de administración profesional para gestionar las encuestas.

🎯 Votación

Permite a los usuarios votar en diferentes preguntas y ver los resultados actualizados.

⚙️ Panel Admin

Interfaz de administración segura para gestionar preguntas y opciones de respuesta.

📊 Resultados

Visualización de resultados de votación con estadísticas en tiempo real.
